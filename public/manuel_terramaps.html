﻿# 📖 Manuel d'utilisation TerraMaps v2.0

🌐 Accès

  • URL : https://terramaps.vercel.app
  • Login : admin@terramaps.ma / Admin2026!

    • 🏠 1. TABLEAU DE BORD (Dashboard)

    La page principale affiche :

  • KPIs : Total projets, points topo, alignements
  • Graphiques : Activité mensuelle, volumes
  • Projets récents : Accès rapide
  • Actions rapides : Nouveau projet, import, calculs

    • 📁 2. PROJETS

    Créer un projet

  • Cliquer "Nouveau Projet"
  • Remplir : Nom, Type, Client, Localisation, EPSG
  • Cliquer "Enregistrer"

    • Dans un projet vous avez :

  • Vue générale : Infos du projet
  • Points topo : Liste et carte des points
  • Alignements : Axes routiers
  • MNT : Modèle Numérique de Terrain
  • Volumes : Calcul cubatures
  • Photos : Galerie photos terrain
  • Calques : Gestion des calques
  • Activité : Historique des actions

    • Partager un projet avec un client

  • Ouvrir le projet
  • Cliquer "🔗 Partager"
  • Le lien est copié automatiquement
  • Envoyer le lien au client par WhatsApp/Email
  • Le client voit : nom, points, photos — sans connexion

    • 📍 3. SURVEY POINTS (Points Topographiques)

    Importer des points

  • Aller sur /survey
  • Sélectionner un projet
  • Cliquer "Import CSV"
  • Format : Nom, Code, X, Y, Z

    • Vues disponibles

  • 🗺 Carte : Points sur OpenStreetMap (EPSG:26191)
  • ≡ Tableau : Liste des points avec coordonnées
  • 📊 Graphique : Visualisation canvas
  • 📈 Profil : Profil altimétrique Z

    • Exporter les points

  • Excel : Tableau XLS
  • LandXML : Format standard topo
  • DXF : AutoCAD/Covadis
  • ZIP : Tous les formats

    • 📄 4. DEVIS

    Créer un devis

  • Aller sur /devis
  • Le numéro DEV-2026-XXX est automatique
  • Remplir : Client, Adresse, Téléphone, Email
  • Sélectionner le projet
  • Ajouter les prestations (ou utiliser un modèle rapide)
  • Cliquer "Generer le Devis PDF"
  • PDF téléchargé automatiquement

    • Modèles rapides disponibles

  • Levé standard < 1 Ha (3 500 MAD)
  • Levé 1-5 Ha + rapport (5 500 MAD)
  • Polygonale complète (4 000 MAD)
  • Bornage terrain (4 050 MAD)

    • Partager un devis avec le client

  • Générer le devis PDF
  • Cliquer "🔗 Partager le dernier devis"
  • Lien copié → envoyer au client
  • Le client voit le devis en ligne sans connexion

    • 🧾 5. FACTURES

    Créer une facture

  • Aller sur /facture
  • Le numéro FAC-2026-XXX est automatique
  • Remplir les informations client
  • Ajouter les prestations
  • Signer avec le pad de signature
  • Cliquer "🧾 Générer la Facture PDF"
  • PDF avec timbre fiscal 20 MAD inclus

    • Partager une facture

  • Générer la facture
  • Cliquer "🔗 Partager la derniere facture"
  • Envoyer le lien au client

    • 📸 6. PHOTOS TERRAIN

    Uploader des photos

  • Aller sur /photos ou onglet Photos dans le projet
  • Sélectionner le projet
  • Choisir une photo depuis votre appareil
  • Ajouter une description (optionnel)
  • Cliquer "📤 Uploader"

    • Les photos sont visibles par le client

    Quand vous partagez un projet, le client voit aussi les photos terrain !

    📅 7. PLANNING MISSIONS

    Créer une mission

  • Aller sur /planning
  • Cliquer "+ Nouvelle mission"
  • Remplir : Titre, Projet, Technicien, Date
  • Cliquer "Créer"

    • Envoyer un rappel email

  • Cliquer 📧 sur la mission
  • Entrer l'email du technicien
  • Cliquer OK → Email envoyé automatiquement

    • 📊 8. LEVÉ TOPOGRAPHIQUE OFFICIEL

    Générer un levé officiel PDF

  • Aller sur /leve-topo
  • Remplir : Province, Commune, Propriétaire, CIN
  • Sélectionner le projet avec les points
  • Dessiner la signature
  • Cliquer "Générer le Levé PDF"

    • Envoyer par email au client

  • Entrer l'email du client
  • Cliquer "📧 Envoyer"
  • Le client reçoit les détails par email

    • ⚙️ 9. PARAMÈTRES SOCIÉTÉ

    Configurer votre société

  • Aller sur /settings
  • Remplir : Nom société, RC, IF, ICE
  • Uploader votre logo (affiché sur tous les PDFs)
  • Sauvegarder

    • 🔐 10. SÉCURITÉ

    Activer le 2FA (Double authentification)

  • Aller sur /profile
  • Cliquer "🔐 Configurer le 2FA"
  • Scanner le QR Code avec Google Authenticator
  • Entrer le code à 6 chiffres
  • ✅ 2FA activé !

    • 📱 11. INSTALLATION MOBILE (PWA)

    Android

  • Ouvrir terramaps.vercel.app dans Chrome
  • Menu ⋮ → "Ajouter à l'écran d'accueil"
  • Confirmer → Icône TerraMaps installée !

    • iPhone

  • Ouvrir dans Safari
  • Bouton Partager → "Sur l'écran d'accueil"
  • Confirmer !

    • 🔔 12. NOTIFICATIONS PUSH

  • Aller sur /profile
  • Cliquer "🔕 Activer notifs"
  • Accepter la permission
  • ✅ Vous recevrez des notifications même hors ligne

    • 💰 13. FINANCE

    Tableau de bord financier

  • Aller sur /finance
  • Voir : Revenus, Factures payées/en attente
  • Cliquer "📄 Rapport mensuel PDF"

    • 🗺️ 14. OUTILS TOPOGRAPHIQUES

    | Page | Outil | |------|-------| | /nivellement | Circuit de nivellement | | /polygonale | Calcul de polygonale | | /calculatrice | Calculatrice topo | | /volumes | Cubatures déblai/remblai | | /alignment | Axes routiers | | /terrain | Modèle numérique de terrain | | /canvas | Dessin CAO |

    📦 15. EXPORT & BACKUP

  • Aller sur /export
  • Choisir : Backup complet, Projets, Devis, ou Factures
  • Fichier JSON téléchargé
